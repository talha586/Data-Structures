Data Structures

This repository contains implementations and coursework from the Data Structures course at **FAST NUCES**.

📖 About This Course

Data Structures focuses on designing efficient ways to store, organise, and manipulate data. The course covers both theory and hands-on implementation of fundamental data structures and their real-world applications.

🗂️ Topics Covered

- Arrays and linked lists (singly, doubly, circular)
- Stacks and queues
- Trees (binary tree, BST, AVL tree)
- Heaps and priority queues
- Hashing and hash tables
- Graphs and graph traversals (BFS, DFS)
- Sorting algorithms (merge sort, quick sort, heap sort)

## 🛠️ Language Used

C++

🚀 How to Run

1. Clone the repository
   ```bash
   git clone https://github.com/talha586/Data-Structures.git
   ```
2. Open any `.cpp` file in your IDE
3. Compile and run:
   ```bash
   g++ filename.cpp -o output
   ./output
   ```
